# 해외연구인력 통합지원 플랫폼 — 디자인 시안

## 프로젝트 개요

**사업명**: 2026년 해외연구인력 생활 및 경력성장 통합지원 플랫폼 구축  
**목적**: 제안서 작성을 위한 메인 페이지 2종 + 서브 페이지 6종 디자인 시안  
**커버리지**: RFP 기능 요구사항 SFR-001 ~ SFR-013 전범위

## 공통 디자인 시스템

**파일**: `css/common.css`

- Secondary: `#00A878` (그린)
- Accent: `#F5A623` (오렌지)
- Danger: `#E53E3E`
- Background: `#F7F9FC`
